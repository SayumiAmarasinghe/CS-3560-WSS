## Getting Started

Welcome to the VS Code Java world. Here is a guideline to help you get started to write Java code in Visual Studio Code.

## Folder Structure

The workspace contains three folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies
- `gui`: the folder to maintain UI components

Meanwhile, the compiled output files will be generated in the `bin` folder by default.

> If you want to customize the folder structure, open `.vscode/settings.json` and update the related settings there.

## How do I compile/run this project?

1. Make sure you are in the `/WSS` directory

2. Run the script to Compile/Run the program

- MAC user:

```bash
./run.sh
```

- WINDOWS powershell user:

```powershell
./run.ps1
```

This will generate java bytecode files (\*.class) in the directory, `/WSS/bin`, which should be git-ignored.

Then, it executes `Main.java`, our application GUI.
